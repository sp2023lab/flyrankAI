Added a Groq-backed report summarisation feature to the existing FastAPI PDF report generator. The POST /api/v1/summarize endpoint returns exactly three schema-validated bullets. The AI provider is isolated behind a provider-neutral interface, every request has an explicit timeout, 429 and 5xx responses are retried with short exponential backoff, 400 responses are not retried, malformed output is retried once and handled safely, and every successful call logs input/output token counts plus an estimated USD cost. The API key is loaded from .env and documented through .env.example; prompts, source text and secrets are not logged. Tests use mocked providers/transports and do not consume API quota.
